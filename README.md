## Setup

1. Clone the repository

2. Install dependencies

```bash
go mod tidy
```

3. Start MySQL

```bash
docker compose up -d
```

4. Run the API server

```bash
go run .
```

## Development

The API runs on `http://localhost:4649` by default.
